# modeling-workshop

Demo repo for Concerto modeling workshop
For concerto docs, visit https://concerto.accordproject.org/

1. Clone this repo to your local machine or use VS Code in the browser at https://vscode.dev/github/DianaLease/modeling-workshop-1
2. Install recommended extensions: Accord Project (Concerto) and Markdown Preview Mermaid Support
3. See examples in the `models` directory
4. Create a new cto file with your own model
5. Right-click a model file in the VS Code Explorer and choose "Compile Model to Target" and choose "markdown"
6. A new directory called `output` will be created with your markdown file. Right-click the file and choose "Open Preview"
